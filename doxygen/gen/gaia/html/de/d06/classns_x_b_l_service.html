This method loads a binding doc and then builds the specific binding required. It can also peek without building.

引数:
aBoundElementthe element to get a binding for
aURIthe binding URI
aPeekFlagif true then just peek to see if the binding is ready
aIsReady[out] if the binding is ready or not
aResult[out] where to store the resulting binding (not used if aPeekFlag is true, otherwise it must be non-null)
aDontExtendURIsa set of URIs that are already bound to this element. If a binding extends any of these then further loading is aborted (because it would lead to the binding extending itself) and NS_ERROR_ILLEGAL_VALUE is returned.
覚え書き:
This method always calls LoadBindingDocumentInfo(), so it's enough to funnel all security checks through that function.
