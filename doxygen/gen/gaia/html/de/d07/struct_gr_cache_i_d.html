説明

GrCacheID is used create and find cached GrResources (e.g. GrTextures). The ID has two parts: the domain and the key. Domains simply allow multiple clients to use 0-based indices as their cache key without colliding. The key uniquely identifies a GrResource within the domain. Users of the cache must obtain a domain via GenerateDomain().
