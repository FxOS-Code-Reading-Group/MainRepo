説明

Represent a single tile in tiled buffer. The buffer keeps tiles, each tile keeps a reference to a texture client. The texture client is backed by a gfxReusableSurfaceWrapper that implements a copy-on-write mechanism while locked. The tile should be locked before being sent to the compositor and unlocked as soon as it is uploaded to prevent a copy. Ideal place to store per tile debug information.
