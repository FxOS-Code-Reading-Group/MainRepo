attribute wstring mozISpellCheckingEngine::dictionary

The name of the current dictionary. Is either a value from getDictionaryList or the empty string if no dictionary is selected. Setting this attribute to a value not in getDictionaryList will throw NS_ERROR_FILE_NOT_FOUND.

The spellcheck engine will send a notification with "spellcheck-dictionary-update" as topic when this changes. If the dictionary is changed to no dictionary (the empty string), an observer is allowed to set another dictionary before it returns.
