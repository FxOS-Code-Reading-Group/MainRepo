説明

Frame manager interface. The frame manager serves two purposes: provides a service for mapping from content to frame and from out-of-flow frame to placeholder frame. handles structural modifications to the frame model. If the frame model lock can be acquired, then the changes are processed immediately; otherwise, they're queued and processed later.

Do not add virtual methods to this class, or bryner will punish you.
