NS_IMETHOD nsIEditorStyleSheets::ReplaceOverrideStyleSheet ( const nsAString &  aURL) [pure virtual]
void nsIEditorStyleSheets::replaceOverrideStyleSheet ( in AString  aURL)

Load and apply the override style sheet, specified by aURL, to the editor's document, replacing the last override style sheet added (if any). This is always synchronous, so aURL should be a local file with only local . This action is not undoable. It is not intended for "user" style sheets, only for editor developers to add sheets to change display behavior for editing (like showing special cursors) that will not be affected by loading "document" style sheets with addStyleSheet or especially replaceStyleSheet.

引数:
aURLThe style sheet to be loaded and applied.
