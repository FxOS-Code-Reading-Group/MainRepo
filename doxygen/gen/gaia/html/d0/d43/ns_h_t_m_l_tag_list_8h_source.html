00008 /******
00009 
00010   This file contains the list of all HTML tags.
00011   See nsHTMLTags.h for access to the enum values for tags.
00012 
00013   It is designed to be used as inline input to nsHTMLTags.cpp and
00014   nsHTMLContentSink *only* through the magic of C preprocessing.
00015 
00016   All entries must be enclosed in the macro HTML_TAG which will have cruel
00017   and unusual things done to it.
00018 
00019   It is recommended (but not strictly necessary) to keep all entries
00020   in alphabetical order.
00021 
00022   The first argument to HTML_TAG is both the enum identifier of the
00023   property and the string value. The second argument is the "creator"
00024   method of the form NS_New$TAGNAMEElement, that will be used by
00025   nsHTMLContentSink.cpp to create a content object for a tag of that
00026   type. Use NOTUSED, if the particular tag has a non-standard creator.
00027 
00028   The HTML_OTHER macro is for values in the nsHTMLTag enum that are
00029   not strictly tags.
00030 
00031   Entries *must* use only lowercase characters.
00032 
00033   Don't forget to update /editor/libeditor/html/nsHTMLEditUtils.cpp as well.
00034 
00035   ** Break these invariants and bad things will happen. **
00036 
00037  ******/
