説明

An interface describing a global application service that handles performing background update checks and provides utilities for selecting and downloading update patches.

readonly attribute boolean nsIApplicationUpdateService::isOtherInstanceHandlingUpdates

Whether or not a different instance is handling updates of this installation. This currently only ever returns true on Windows when 2 instances of an application are open or when both the Metro and Desktop browsers are open. Only one of the instances will actually handle updates for the installation.
