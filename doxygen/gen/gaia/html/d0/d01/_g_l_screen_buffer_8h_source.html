00006 /* GLScreenBuffer is the abstraction for the "default framebuffer" used
00007  * by an offscreen GLContext. Since it's only for offscreen GLContext's,
00008  * it's only useful for things like WebGL, and is NOT used by the
00009  * compositor's GLContext. Remember that GLContext provides an abstraction
00010  * so that even if you want to draw to the 'screen', even if that's not
00011  * actually the screen, just draw to 0. This GLScreenBuffer class takes the
00012  * logic handling out of GLContext.
00013 */
