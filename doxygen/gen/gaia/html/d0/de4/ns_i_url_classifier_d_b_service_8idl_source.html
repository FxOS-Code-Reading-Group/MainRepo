00112   // Incremental update methods.
00113   //
00114   // An update to the database has the following steps:
00115   //
00116   // 1) The update process is started with beginUpdate().  The client
00117   //    passes an nsIUrlClassifierUpdateObserver object which will be
00118   //    notified as the update is processed by the dbservice.
00119   // 2) The client sends an initial update stream to the dbservice,
00120   //    using beginStream/updateStream/finishStream.
00121   // 3) While reading this initial update stream, the dbservice may
00122   //    request additional streams from the client as requested by the
00123   //    update stream.
00124   // 4) For each additional update stream, the client feeds the
00125   //    contents to the dbservice using beginStream/updateStream/endStream.
00126   // 5) Once all streams have been processed, the client calls
00127   //    finishUpdate.  When the dbservice has finished processing
00128   //    all streams, it will notify the observer that the update process
00129   //    is complete.
00130
