bool GrResource::isValid ( ) const

Tests whether a resource has been abandoned or released. All resources will be in this state after their creating GrContext is destroyed or has contextLost called. It's up to the client to test isValid() before attempting to use a resource if it holds refs on resources across ~GrContext, freeResources with the force flag, or contextLost.

戻り値:
true if the resource has been released or abandoned, false otherwise.
