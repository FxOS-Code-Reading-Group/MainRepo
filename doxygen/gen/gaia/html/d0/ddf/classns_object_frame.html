void nsObjectFrame::SetEmptyWidgetConfiguration ( )

Set the next widget configuration for the plugin to the desired position of the plugin's widget, on the assumption that it is not visible (clipped out or covered by opaque content). This will only be called for plugins which have been registered with the root pres context for geometry updates. If there is no widget associated with the plugin, this will have no effect.
