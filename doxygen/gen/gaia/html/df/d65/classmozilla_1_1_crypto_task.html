説明

Frequently we need to run a task on a background thread without blocking the main thread, and then call a callback on the main thread with the result. This class provides the framework for that. Subclasses must:

(1) Override CalculateResult for the off-the-main-thread computation. NSS functionality may only be accessed within CalculateResult. (2) Override ReleaseNSSResources to release references to all NSS resources (that do implement nsNSSShutDownObject themselves). (3) Override CallCallback() for the on-the-main-thread call of the callback.

CalculateResult, ReleaseNSSResources, and CallCallback are called in order, except CalculateResult might be skipped if NSS is shut down before it can be called; in that case ReleaseNSSResources will be called and then CallCallback will be called with an error code.

virtual void mozilla::CryptoTask::ReleaseNSSResources ( ) [protected, pure virtual]

Called on the main thread during NSS shutdown or just before CallCallback has been called. All NSS resources must be released. Usually, this just means assigning nullptr to the ScopedNSSType-based memory variables.
