説明

template<typename E, E smallestLegal, E highBound>
struct IPC::EnumSerializer< E, smallestLegal, highBound >

Generic enum serializer.

This is a generic serializer for any enum type used in IPDL. Programmers can define ParamTraits<E> for enum type E by deriving EnumSerializer<E, smallestLegal, highGuard>.

The serializer would check value againts a range specified by smallestLegal and highGuard. Only values from smallestLegal to highGuard are valid, include smallestLegal but highGuard.

For example, following is definition of serializer for enum type FOO. 

 enum FOO { FOO_FIRST, FOO_SECOND, FOO_LAST, NUM_FOO };

 template <>
 struct ParamTraits<FOO>:
     public EnumSerializer<FOO, FOO_FIRST, NUM_FOO> {};

FOO_FIRST, FOO_SECOND, and FOO_LAST are valid value.
