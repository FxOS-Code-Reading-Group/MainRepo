uint32_t mozilla::layers::DataTextureSource::GetUpdateSerial ( ) const

A facility to avoid reuploading when it is not necessary. The caller of Update can use GetUpdateSerial to see if the number has changed since last update, and call SetUpdateSerial after each successful update. The caller is responsible for managing the update serial except when the texture data is deallocated in which case the TextureSource should always reset the update serial to zero.
