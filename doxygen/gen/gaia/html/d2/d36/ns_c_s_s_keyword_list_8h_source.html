00006 /* keywords used within CSS property values */
00007 
00008 /******
00009 
00010   This file contains the list of all parsed CSS keywords
00011   See nsCSSKeywords.h for access to the enum values for keywords
00012 
00013   It is designed to be used as inline input to nsCSSKeywords.cpp *only*
00014   through the magic of C preprocessing.
00015 
00016   All entries must be enclosed in the macro CSS_KEY which will have cruel
00017   and unusual things done to it
00018 
00019   It is recommended (but not strictly necessary) to keep all entries
00020   in alphabetical order
00021 
00022   Requirements:
00023 
00024   Entries are in the form: (name, id). 'id' must always be the same as 'name'
00025   except that all hyphens ('-') in 'name' are converted to underscores ('_')
00026   in 'id'. This lets us do nice things with the macros without having to
00027   copy/convert strings at runtime.
00028 
00029   'name' entries *must* use only lowercase characters.
00030 
00031   ** Break these invariants and bad things will happen. **
00032 
00033  ******/
