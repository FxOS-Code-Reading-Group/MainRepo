nsresult mozilla::OggWriter::GetContainerData ( nsTArray< nsTArray< uint8_t > > *  aOutputBufs,
uint32_t  aFlags = 0 
) [virtual]

Copies the final container data to a buffer if it has accumulated enough packets from WriteEncodedTrack. This buffer of data is appended to aOutputBufs, and existing elements of aOutputBufs should not be modified. aFlags is true with FLUSH_NEEDED will force OggWriter to flush an ogg page even it is not full, and copy these container data to a buffer for aOutputBufs to append.

int  aDuration,
uint32_t  aFlags = 0 
) [virtual]

Writes encoded track data from aBuffer to a packet, and insert this packet into the internal stream of container writer. aDuration is the playback duration of this packet in number of samples. aFlags is true with END_OF_STREAM if this is the last packet of track. Currently, WriteEncodedTrack doesn't support multiple tracks.
