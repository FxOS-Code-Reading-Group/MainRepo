enum nsNavigationDirection

nsNavigationDirection: an enum expressing navigation through the menus in terms which are independent of the directionality of the chrome. The terminology, derived from XSL-FO and CSS3 (e.g. http://www.w3.org/TR/css3-text/#TextLayout), is BASE (Before, After, Start, End), with the addition of First and Last (mapped to Home and End respectively).

In languages such as English where the inline progression is left-to-right and the block progression is top-to-bottom (lr-tb), these terms will map out as in the following diagram

--- inline progression --->

First | ... | Before | +--------+ block Start | | End progression +--------+ | After | ... | Last V
