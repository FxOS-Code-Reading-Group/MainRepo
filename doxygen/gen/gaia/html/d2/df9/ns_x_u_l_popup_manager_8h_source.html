00309   // given a menu frame, find the prevous or next menu frame. If aPopup is
00310   // true then navigate a menupopup, from one item on the menu to the previous

00312   // menu. If aIsPopup is false, the navigation is on a menubar, so navigate
00313   // between menus on the menubar. This is used for left/right cursor navigation.
