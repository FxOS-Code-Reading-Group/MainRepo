説明

A MediaSegment is a chunk of media data sequential in time. Different types of data have different subclasses of MediaSegment, all inheriting from MediaSegmentBase. All MediaSegment data is timed using TrackTicks. The actual tick rate is defined on a per-track basis. For some track types, this can be a fixed constant for all tracks of that type (e.g. 1MHz for video).

Each media segment defines a concept of "null media data" (e.g. silence for audio or "no video frame" for video), which can be efficiently represented. This is used for padding.
