00001 # Contributing to Gaia
00002 
00003 Hello!
00004 Thank you for your interests on contributing to the Gaia repository.
00005 
00006 Before submitting a pull request on GitHub, please create a bug report on [Bugzilla](https://bugzilla.mozilla.org/) under the [Boot2Gecko > Gaia](https://bugzilla.mozilla.org/enter_bug.cgi?product=Boot2Gecko&component=Gaia) component.
00007 You should select a more specific component of Boot2Gecko when appropriate (i.e. if your problem concerns a single application).
00008 
00009 Please note that we have rules in place to keep our coding style in sync and our commit history clean.
00010 Be sure that your commit complies with these guidelines, or your change may not be accepted.
00011 Please refer to [the Gaia/Hacking page on MDN](https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Platform/Gaia/Hacking), specifically on the [Submitting a patch section](https://developer.mozilla.org/en-US/docs/Mozilla/Firefox_OS/Platform/Gaia/Hacking#Submitting_a_patch) for complete information.
