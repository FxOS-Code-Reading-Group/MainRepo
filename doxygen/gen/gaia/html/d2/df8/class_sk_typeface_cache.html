型定義

typedef bool(* SkTypefaceCache::FindProc)(SkTypeface *, SkTypeface::Style, void *context)

Callback for FindByProc. Returns true if the given typeface is a match for the given context. The passed typeface is owned by the cache and is not additionally ref()ed. The typeface may be in the disposed state.

static SkTypeface* SkTypefaceCache::FindByID ( SkFontID  fontID) [static]

Search the cache for a typeface with the specified fontID (uniqueID). If one is found, return it (its reference count is unmodified). If none is found, return NULL. The reference count is unmodified as it is assumed that the stack will contain a ref to the typeface.

Iterate through the cache, calling proc(typeface, ctx) with each typeface. If proc returns true, then we return that typeface (this ref()s the typeface). If it never returns true, we return NULL.

Helper: returns a unique fontID to pass to the constructor of your subclass of SkTypeface

static void SkTypefaceCache::PurgeAll ( ) [static]

This will unref all of the typefaces in the cache for which the cache is the only owner. Normally this is handled automatically as needed. This function is exposed for clients that explicitly want to purge the cache (e.g. to look for leaks).
