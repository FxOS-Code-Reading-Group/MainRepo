Getters like ValueFor(aProperty), but for each property by name (ValueForBackgroundColor, etc.), and more efficient than ValueFor. These use the names used for the property on DOM interfaces (the 'method' field in nsCSSPropList.h).

Like ValueFor(), the caller must check that the property is within mSIDs.
