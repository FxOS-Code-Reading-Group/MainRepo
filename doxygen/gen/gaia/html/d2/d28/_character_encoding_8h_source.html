00102  * 2-byte-at-a-time view of a UTF-16 byte stream. This is similar to UCS-2,
00103  * but unlike UCS-2, we do not strip UTF-16 extension bytes. This allows a
00104  * sufficiently dedicated JavaScript program to be fully unicode-aware by
00105  * manually interpreting UTF-16 extension characters embedded in the JS
00106  * string.
00107  */
