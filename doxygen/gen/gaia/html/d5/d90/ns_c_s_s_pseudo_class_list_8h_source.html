00006 /* atom list for CSS pseudo-classes */

00008 /*
00009  * This file contains the list of nsIAtoms and their values for CSS
00010  * pseudo-classes.  It is designed to be used as inline input to
00011  * nsCSSPseudoClasses.cpp *only* through the magic of C preprocessing.
00012  * All entries must be enclosed in the macros CSS_PSEUDO_CLASS,
00013  * CSS_STATE_DEPENDENT_PSEUDO_CLASS, or CSS_STATE_PSEUDO_CLASS which
00014  * will have cruel and unusual things done to them.  The entries should
00015  * be kept in some sort of logical order.  The first argument to
00016  * CSS_PSEUDO_CLASS is the C++ identifier of the atom.  The second
00017  * argument is the string value of the atom.  The third argument is the
00018  * name of the preference controlling whether the pseudo-class is
00019  * recognized by the parser, or the empty string if it's unconditional.
00020  *
00021  * CSS_STATE_DEPENDENT_PSEUDO_CLASS and CSS_STATE_PSEUDO_CLASS also take
00022  * the name of the state bits that the class corresponds to.  Only one
00023  * of the bits needs to match for a CSS_STATE_PSEUDO_CLASS to match;
00024  * CSS_STATE_DEPENDENT_PSEUDO_CLASS matching depends on a customized per-class
00025  * algorithm which should be defined in SelectorMatches() in
00026  * nsCSSRuleProcessor.cpp.
00027  *
00028  * If CSS_STATE_PSEUDO_CLASS is not defined, it'll be automatically
00029  * defined to CSS_STATE_DEPENDENT_PSEUDO_CLASS;
00030  * if CSS_STATE_DEPENDENT_PSEUDO_CLASS is not defined, it'll be
00031  * automatically defined to CSS_PSEUDO_CLASS.
00032  */
