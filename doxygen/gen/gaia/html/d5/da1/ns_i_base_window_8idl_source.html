00027     /*
00028     Allows a client to initialize an object implementing this interface with
00029     the usually required window setup information.
00030     It is possible to pass null for both parentNativeWindow and parentWidget,
00031     but only docshells support this.
00032 
00033     @param parentNativeWindow - This allows a system to pass in the parenting
00034         window as a native reference rather than relying on the calling
00035         application to have created the parent window as an nsIWidget.  This 
00036         value will be ignored (should be nullptr) if an nsIWidget is passed in to
00037         the parentWidget parameter.
00038 
00039     @param parentWidget - This allows a system to pass in the parenting widget.
00040         This allows some objects to optimize themselves and rely on the view
00041         system for event flow rather than creating numerous native windows.  If
00042         one of these is not available, nullptr should be passed.
00043 
00044     @param x - This is the x co-ordinate relative to the parent to place the
00045         window.
00046 
00047     @param y - This is the y co-ordinate relative to the parent to place the 
00048         window.
00049 
00050     @param cx - This is the width   for the window to be.
00051 
00052     @param cy - This is the height for the window to be.
00053 
00054     @return NS_OK - Window Init succeeded without a problem.
00055                 NS_ERROR_UNEXPECTED - Call was unexpected at this time.  Most likely
00056                     due to you calling it after create() has been called.
00057                 NS_ERROR_INVALID_ARG - controls that require either a parentNativeWindow
00058                     or a parentWidget may return invalid arg when they do not 
00059                     receive what they are needing.
00060     */
00061     [noscript]void initWindow(in nativeWindow parentNativeWindow, 
00062         in nsIWidget parentWidget,  in long x, in long y, in long cx, in long cy);
00063 
00064     /*
00065     Tells the window that intialization and setup is complete.  When this is
00066     called the window can actually create itself based on the setup
00067     information handed to it.
00068 
00069     @return NS_OK - Creation was successfull.
00070                 NS_ERROR_UNEXPECTED - This call was unexpected at this time.
00071                     Perhaps create() had already been called or not all
00072                     required initialization had been done.
