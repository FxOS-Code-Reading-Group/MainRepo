int GrTextureDesc::fSampleCnt

The number of samples per pixel or 0 to disable full scene AA. This only applies if the kRenderTarget_GrTextureFlagBit is set. The actual number of samples may not exactly match the request. The request will be rounded up to the next supported sample count, or down if it is larger than the max supported count.
