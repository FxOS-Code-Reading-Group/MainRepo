enum NSPluginCallReentry

Used to indicate whether it's OK to reenter Gecko and repaint, flush frames, run scripts, etc, during this plugin call. When NS_PLUGIN_CALL_UNSAFE_TO_REENTER_GECKO is set, we try to avoid dangerous Gecko activities when the plugin spins a nested event loop, on a best-effort basis.
