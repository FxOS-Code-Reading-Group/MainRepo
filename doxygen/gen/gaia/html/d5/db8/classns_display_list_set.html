説明

This is passed as a parameter to nsIFrame::BuildDisplayList. That method will put any generated items onto the appropriate list given here. It's basically just a collection with one list for each separate stacking layer. The lists themselves are external to this object and thus can be shared with others. Some of the list pointers may even refer to the same list.
