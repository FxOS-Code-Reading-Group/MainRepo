readonly attribute jsval nsITelemetry::addonHistogramSnapshots

An object containing a snapshot from all of the currently registered addon histograms. { addon-id1 : data1, ... }

where data is an object whose properties are the names of the addon's histograms and whose corresponding values are as in histogramSnapshots.
