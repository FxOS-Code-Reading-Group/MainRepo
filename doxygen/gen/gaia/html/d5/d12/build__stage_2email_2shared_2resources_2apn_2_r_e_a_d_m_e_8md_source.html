00001 Three APN (Access Point Name) databases are used:
00002 
00003 * the Android database `/shared/resources/apn/apns_conf.xml` is the *de facto* standard we rely on;
00004 * a local database `/shared/resources/apn/apns_conf-local.xml` with the same format we merge with the above, used for where Google is lacking. Ideally this file should be empty as everything should be put upstream;
00005 * the Gnome database `/shared/resources/apn/service_providers.xml` is probably less up-to-date but has a bit more information.
00006 
00007 A fourth file `/shared/resources/apn/operator-variant.xml` is also used.
00008 
00009 These files are only used to test and merge these APN databases.
00010 
00011 The `/shared/resources/apn.json` file is a JSON version of the Android database, with a few additional informations from the Gnome APN database (e.g.  voicemail number).
00012 
00013 To regenerate the .json file, delete the apn.json first, then start a webserver from gaia/shared/resource folder. Like this:
00014 
00015   python -m SimpleHTTPServer 4104
00016 
00017 Then load http://0.0.0.0:4104/apn
00018 This should regenerate the .json file into the left part of the page, and you can cut & paste this to your editor.
