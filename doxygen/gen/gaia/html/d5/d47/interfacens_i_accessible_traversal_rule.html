void nsIAccessibleTraversalRule::getMatchRoles ( [array, size_is(aCount)] out unsigned long  aRoles,
[retval] out unsigned long  aCount 
)

Retrieve a list of roles that the traversal rule should test for. Any node with a role not in this list will automatically be ignored. An empty list will match all roles. It should be assumed that this method is called once at the start of a traversal, so changing the method's return result after that would have no affect.

引数:
aRoles[out] an array of the roles to match.
aCount[out] the length of the array.
