virtual bool mozilla::layers::Composer2D::TryRender ( Layer *  aRoot,
const gfxMatrix &  aWorldTransform 
) [pure virtual]

Return true if |aRoot| met the implementation's criteria for fast composition and the render was successful. Return false to fall back on the GPU.

|aWorldTransform| must be applied to |aRoot|'s subtree when rendering to the framebuffer. This is a global transform on the entire scene, defined in GL space. If the Composer2D implementation is unable to honor the transform, it should return false.

Currently, when TryRender() returns true, the entire framebuffer must have been rendered.
