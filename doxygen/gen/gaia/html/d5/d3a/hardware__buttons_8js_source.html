00001 // hardware_buttons.js:
00002 //
00003 // Gecko code in b2g/chrome/content/shell.js sends mozChromeEvents
00004 // when the user presses or releases a hardware button such as Home, Sleep,
00005 // and Volume Up and Down.
00006 //
00007 // This module listens for those low-level mozChromeEvents, processes them
00008 // and generates higher-level events to handle autorepeat on the volume keys
00009 // long presses on Home and Sleep, and the Home+Sleep key combination.
00010 //
00011 // Other system app modules should listen for the high-level button events
00012 // generated by this module.
00013 //
00014 // The low-level input events processed by this module have type set
00015 // to "mozChromeEvent" and detail.type set to one of:
00016 //
00017 //   home-button-press
00018 //   home-button-release
00019 //   sleep-button-press
00020 //   sleep-button-release
00021 //   volume-up-button-press
00022 //   volume-up-button-release
00023 //   volume-down-button-press
00024 //   volume-down-button-release
00025 //
00026 // The high-level events generated by this module are simple Event objects
00027 // that are not cancelable and do not bubble.  The are dispatched at the
00028 // window object.  The type property is set to one of these:
00029 //
00030 // Event Type    Meaning
00031 // --------------------------------------------------------------
00032 //   home        short press and release of home button
00033 //   holdhome    long press and hold of home button
00034 //   sleep       short press and release of sleep button
00035 //   wake        sleep or home pressed while sleeping
00036 //   holdsleep   long press and hold of sleep button
00037 //   volumeup    volume up pressed and released or autorepeated
00038 //   volumedown  volume down pressed and released or autorepeated
00039 //   home+sleep  home and sleep pressed at same time (used for screenshots)
00040 //
00041 // Because these events are fired at the window object, they cannot be
00042 // captured.  Many modules listen for the home event. Those that want
00043 // to respond to it and prevent others from responding should call
00044 // stopImmediatePropagation(). Overlays that want to prevent the window
00045 // manager from showing the homescreen on the home event should call that
00046 // method.  Note, however, that this only works for scripts that run and
00047 // register their event handlers before window_manager.js does.
00048 //
