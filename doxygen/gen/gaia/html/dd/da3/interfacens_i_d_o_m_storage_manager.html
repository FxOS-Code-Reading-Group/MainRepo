説明

General purpose interface that has two implementations, for localStorage resp. sessionStorage with "@mozilla.org/dom/localStorage-manager;1" resp. "@mozilla.org/dom/sessionStorage-manager;1" contract IDs.

Returns instance of localStorage object for aURI's origin. This method ensures there is always only a single instance for a single origin.

Currently just forwards to the createStorage method of this interface.

Extension developers are strongly encouraged to use getStorage or createStorage method instead.
