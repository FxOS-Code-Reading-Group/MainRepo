Create a frame iterator with the specified properties.

引数:
aEnumerator[out] the created iterator
aPresContext[in]
aStart[in] the frame to start iterating from
aType[in] the type of the iterator: leaf, pre-order, or post-order
aVisual[in] whether the iterator should traverse frames in visual bidi order
aLockInScrollView[in] whether to stop iterating when exiting a scroll view
aFollowOOFs[in] whether the iterator should follow out-of-flows. If true, when reaching a placeholder frame while going down will get the real frame. Going back up will go on past the placeholder, so the placeholders are logically part of the frame tree.
