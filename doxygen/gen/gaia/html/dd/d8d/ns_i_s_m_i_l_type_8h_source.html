00014 // nsISMILType: Interface for defining the basic operations needed for animating
00015 // a particular kind of data (e.g. lengths, colors, transformation matrices).
00016 //
00017 // This interface is never used directly but always through an nsSMILValue that
00018 // bundles together a pointer to a concrete implementation of this interface and
00019 // the data upon which it should operate.
00020 //
00021 // We keep the data and type separate rather than just providing different
00022 // subclasses of nsSMILValue. This is so that sizeof(nsSMILValue) is the same
00023 // for all value types, allowing us to have a type-agnostic nsTArray of
00024 // nsSMILValue objects (actual objects, not pointers). It also allows most
00025 // nsSMILValues (except those that need to allocate extra memory for their
00026 // data) to be allocated on the stack and directly assigned to one another
00027 // provided performance benefits for the animation code.
00028 //
00029 // Note that different types have different capabilities. Roughly speaking there
00030 // are probably three main types:
00031 //
00032 // +---------------------+---------------+-------------+------------------+
00033 // | CATEGORY:           | DISCRETE      | LINEAR      | ADDITIVE         |
00034 // +---------------------+---------------+-------------+------------------+
00035 // | Example:            | strings,      | path data?  | lengths,         |
00036 // |                     | color k/words?|             | RGB color values |
00037 // |                     |               |             |                  |
00038 // | -- Assign?          |     X         |    X        |    X             |
00039 // | -- Add?             |     -         |    X?       |    X             |
00040 // | -- SandwichAdd?     |     -         |    -?       |    X             |
00041 // | -- ComputeDistance? |     -         |    -        |    X?            |
00042 // | -- Interpolate?     |     -         |    X        |    X             |
00043 // +---------------------+---------------+-------------+------------------+
00044 //
