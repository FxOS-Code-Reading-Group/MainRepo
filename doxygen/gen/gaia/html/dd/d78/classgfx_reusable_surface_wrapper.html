説明

Provides a cross thread wrapper for a gfxImageSurface that has copy-on-write schemantics.

Only the owner thread can write to the surface and aquire read locks.

OMTC Usage: 1) Content creates a writable copy of this surface wrapper which be optimized to the same wrapper if there are no readers. 2) The surface is sent from content to the compositor once or potentially many time, each increasing a read lock. 3) When the compositor has processed the surface and uploaded the content it then releases the read lock.
