Called when an SSL connection aborts because of a bad certificate, and no other code is in place for reporting the problem. Should bring up a dialog to inform the user and display the certificate.

引数:
statusMight be used to query additional information
certThe certificate that this error is about
textErrorMessageAn error message with whitespace formatting
htmlErrorMessageOptional, might either be empty, or might contain the same message as in textErrorMessage plus some html formatting.
hostNameThe error occurred when connecting to this hostName.
portNumberThe error occurred when connecting to this portNumber.
