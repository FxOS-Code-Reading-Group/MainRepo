attribute ACString updateUrl

説明

This is a class to manage large table updates from the server. Rather than downloading the whole update and then updating the sqlite database, we update tables as the data is streaming in.

Try to download updates from updateUrl. Only one instance of this runs at a time, so we return false if another instance is already running.

引数:
aRequestTablesComma-separated list of tables included in this update.
aRequestBodyThe body for the request.
aClientKeyThe client key for checking the update's MAC.
aSuccessCallbackCalled after a successful update.
aUpdateErrorCallbackCalled for problems applying the update
aDownloadErrorCallbackCalled if we get an http error or a connection refused error.
