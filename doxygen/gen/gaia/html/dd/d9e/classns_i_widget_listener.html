) [virtual]

Called when the z-order of the window is changed. Returns true if the notification was handled. aPlacement indicates the new z order. If placement is nsWindowZRelative, then aRequestBelow should be the window to place below. On return, aActualBelow will be set to the window actually behind. This generally only applies to Windows.
