説明

A view on a TextureHost where the texture is internally represented as tiles (contrast with a tiled buffer, where each texture is a tile). For iteration by the texture's buffer host. This is only useful when the underlying surface is too big to fit in one device texture, which forces us to split it in smaller parts. Tiled Compositable is a different thing.
