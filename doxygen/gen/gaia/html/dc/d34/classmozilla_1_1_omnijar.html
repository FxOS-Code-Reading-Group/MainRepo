nsACString &  result

Returns the URI string corresponding to the omni.jar or directory for GRE or APP. i.e. jar:/path/to/omni.jar!/ for omni.jar and /path/to/base/dir/ otherwise. Returns an empty string for APP in the unified case. The returned URI is guaranteed to end with a slash.

Initializes the Omnijar API with the given directory or file for GRE and APP. Each of the paths given can be:

  • a file path, pointing to the omnijar file,
