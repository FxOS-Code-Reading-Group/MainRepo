nsTDependentSubstring_CharT

A string class which wraps an external array of string characters. It is the client code's responsibility to ensure that the external buffer remains valid for a long as the string is alive.

NAMES: nsDependentSubstring for wide characters nsDependentCSubstring for narrow characters
