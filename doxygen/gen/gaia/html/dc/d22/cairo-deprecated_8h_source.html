00042  * into providing a little bit of guidance with its error
00043  * messages. The idea is to help users port their old code without
00044  * having to dig through lots of documentation.
00045  *
00046  * The first set of REPLACED_BY functions is for functions whose names
00047  * have just been changed. So fixing these up is mechanical, (and
00048  * automated by means of the cairo/util/cairo-api-update script.
00049  *
00050  * The second set of DEPRECATED_BY functions is for functions where
00051  * the replacement is used in a different way, (ie. different
00052  * arguments, multiple functions instead of one, etc). Fixing these up
