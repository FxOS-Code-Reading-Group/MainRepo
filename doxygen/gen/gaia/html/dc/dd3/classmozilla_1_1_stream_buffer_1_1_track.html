説明

Every track has a start time --- when it started in the StreamBuffer. It has an end flag; when false, no end point is known; when true, the track ends when the data we have for the track runs out. Tracks have a unique ID assigned at creation. This allows us to identify the same track across StreamBuffers. A StreamBuffer should never have two tracks with the same ID (even if they don't overlap in time). TODO Tracks can also be enabled and disabled over time.
