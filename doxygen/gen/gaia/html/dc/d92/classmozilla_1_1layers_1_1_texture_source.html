説明

TextureSource is the interface for texture objects that can be composited by a given compositor backend. Since the drawing APIs are different between backends, the TextureSource interface is split into different interfaces (TextureSourceOGL, etc.), and TextureSource mostly provide access to these interfaces.

This class is used on the compositor side.
