説明

This special accessibility class is for the caret and selection management. There is only 1 visible caret per top level window. However, there may be several visible selections.

The important selections are the one owned by each document, and the one in the currently focused control.

On Windows this class is used to move an invisible system caret that shadows the Mozilla caret. Windows will also automatically map this to the MSAA caret accessible object (via OBJID_CARET) (as opposed to the root accessible tree for a window which is retrieved with OBJID_CLIENT).

For ATK and IAccessible2, this class is used to fire caret move and selection change events.
