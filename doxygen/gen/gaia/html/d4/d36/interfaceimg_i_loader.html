)

Start the load and decode of an image.

引数:
aChannelthe channel to load the image from. This must already be opened before ths method is called, and there must have been no OnDataAvailable calls for it yet.
aObserverthe observer (may be null)
cxsome random data
aListener[out] A listener that you must send the channel's notifications and data to. Can be null, in which case imagelib has found a cached image and is not interested in the data. will be canceled for you in this case.

libpr0n does NOT keep a strong ref to the observer; this prevents reference cycles. This means that callers of loadImageWithChannel should make sure to Cancel() the resulting request before the observer goes away.
