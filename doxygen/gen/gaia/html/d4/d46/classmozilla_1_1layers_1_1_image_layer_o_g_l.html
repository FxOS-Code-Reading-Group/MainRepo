virtual bool mozilla::layers::ImageLayerOGL::LoadAsTexture ( GLuint  aTextureUnit,
gfxIntSize *  aSize 
) [virtual]

Loads the result of rendering the layer as an OpenGL texture in aTextureUnit. Will try to use an existing texture if possible, or a temporary one if not. It is the callee's responsibility to release the texture. Will return true if a texture could be constructed and loaded, false otherwise. The texture will not be transformed, i.e., it will be in the same coord space as this. Any layer that can be used as a mask layer should override this method. aSize will contain the size of the image.
