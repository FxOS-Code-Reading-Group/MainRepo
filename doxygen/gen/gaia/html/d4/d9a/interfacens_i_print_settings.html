void nsIPrintSettings::GetPageRanges ( in IntegerArray  aPages)

Get more accurate print ranges from the superior interval (startPageRange, endPageRange). The aPages array is populated with a list of pairs (start, end), where the endpoints are included. The print ranges (start, end), must not overlap and must be in the (startPageRange, endPageRange) scope.

If there are no print ranges the aPages array is cleared.
