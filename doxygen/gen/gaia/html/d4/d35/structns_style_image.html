説明

Represents a paintable image of one of the following types. (1) A real image loaded from an external source. (2) A CSS linear or radial gradient. (3) An element within a document, or an , <video>, or <canvas> element not in a document. (*) Optionally a crop rect can be set to paint a partial (rectangular) region of an image. (Currently, this feature is only supported with an image of type (1)).

This struct is currently used only for 'background-image', but it may be used by other CSS properties such as 'border-image', 'list-style-image', and 'content' in the future (bug 507052).
