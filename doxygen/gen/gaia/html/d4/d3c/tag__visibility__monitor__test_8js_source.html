00002   MonitorTagVisibility Tests
00003 
00004   - implements and runs a DSL, monitoring changes that took place between
00005     instructions
00006 
00007   - DSL
00008       - instructions are either a string, or an array of strings
00009           - instructions are posted to the event loop and run consecutively
00010           - if instruction is an array, subinstructions are run consecutively,
00011             not posted to the event loop consecutively
00012       - instructions access elements with identifiers composed of numbers and
00013         commas
00014       - %d will be used to represent a number
00015       - %i and %j will be used to represent an identifier
00016       - instructions:
00017         scroll %d
00018           sets container's scrolltop to %d
00019         rm %i
00020           removes element %i
00021         addafter %i %j
00022           add %i after %j (so %j's next sibling is %i)
00023         addbefore %i %j
00024           add %i before %j (so %j's prev sibling is %i)
00025         add %i
00026           append %i to container
00027 
00028 ====================================*/
