template<class WebIDLCallbackT, class XPCOMCallbackT>
mozilla::dom::CallbackObjectHolder< WebIDLCallbackT, XPCOMCallbackT >::CallbackObjectHolder ( WebIDLCallbackT *  aCallback) [explicit]

A class which stores either a WebIDLCallbackT* or an XPCOMCallbackT*. Both types must inherit from nsISupports. The pointer that's stored can be null.

When storing a WebIDLCallbackT*, mPtrBits is set to the pointer value. When storing an XPCOMCallbackT*, mPtrBits is the pointer value with low bit set.
