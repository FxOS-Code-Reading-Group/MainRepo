GhostURLsReporter generates the "ghost-windows" multi-report, which includes a list of all ghost windows' URLs. If you're only interested in this list, running this report is faster than running nsWindowMemoryReporter.

nsGhostWindowReporter generates the "ghost-windows" single-report, which counts the number of ghost windows present.
