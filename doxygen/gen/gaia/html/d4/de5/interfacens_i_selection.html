void nsISelection::modify ( in DOMString  alter,
in DOMString  direction,
in DOMString  granularity 
)

Modifies the selection. Note that the parameters are case-insensitive.

引数:
altercan be one of { "move", "extend" }
  • "move" collapses the selection to the end of the selection and applies the movement direction/granularity to the collapsed selection.
  • "extend" leaves the start of the selection unchanged, and applies movement direction/granularity to the end of the selection.
directioncan be one of { "forward", "backward", "left", "right" }
granularitycan be one of { "character", "word", "line", "lineboundary" }
戻り値:
NS_ERROR_NOT_IMPLEMENTED if the granularity is "sentence", "sentenceboundary", "paragraph", "paragraphboundary", or "documentboundary". Returns NS_ERROR_INVALID_ARG if alter, direction, or granularity has an unrecognized value.
