Define typesafe getter functions for each style struct by preprocessing the list of style structs. These functions are the preferred way to get style data. The macro creates functions like: const nsStyleBorder* StyleBorder(); const nsStyleColor* StyleColor();

PeekStyle* is like GetStyle* but doesn't trigger style computation if the data is not cached on either the style context or the rule node.

Perhaps this shouldn't be a public nsStyleContext API.
