説明

Line iterator API.

Lines are numbered from 0 to N, where 0 is the top line and N is the bottom line.

Obtain this interface from frames via nsIFrame::GetLineIterator. When you are finished using the iterator, call DisposeLineIterator() to destroy the iterator if appropriate.
