00009  * nssckfwc.h
00010  *
00011  * This file prototypes all of the NSS Cryptoki Framework "wrapper" 
00012  * which implement the PKCS#11 API.  Technically, these are public
00013  * routines (with capital "NSS" prefixes), since they are called
00014  * from (generated) code within a Module using the Framework.
00015  * However, they should not be called except from those generated
00016  * calls.  Hence, the prototypes have been split out into this file.
