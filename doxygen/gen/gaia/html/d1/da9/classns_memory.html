説明

Static helper routines to manage memory. These routines allow easy access to xpcom's built-in (global) nsIMemory implementation, without needing to go through the service manager to get it. However this requires clients to link with the xpcom DLL.

This class is not threadsafe and is intented for use only on the main thread.
