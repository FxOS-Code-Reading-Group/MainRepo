int32_t mozilla::dom::SVGTests::GetBestLanguagePreferenceRank ( const nsSubstring &  aAcceptLangs) const

Compare the language name(s) in a systemLanguage attribute to the user's language preferences, as defined in http://www.w3.org/TR/SVG11/struct.html#SystemLanguageAttribute We have a match if a language name in the users language preferences exactly equals one of the language names or exactly equals a prefix of one of the language names in the systemLanguage attribute.

戻り値:
2 * the lowest index in the aAcceptLangs that matches + 1 if only the prefix matches, or -1 if no indices match. XXX This algorithm is O(M*N).

bool mozilla::dom::SVGTests::PassesConditionalProcessingTests ( const nsString *  aAcceptLangs = nullptr) const

Check whether the conditional processing attributes requiredFeatures, requiredExtensions and systemLanguage all "return true" if they apply to and are specified on the given element. Returns true if this element should be rendered, false if it should not.

引数:
aAcceptLangsOptional parameter to pass in the value of the intl.accept_languages preference if the caller has it cached. Alternatively, pass in kIgnoreSystemLanguage to skip the systemLanguage check if the caller is giving that special treatment.
