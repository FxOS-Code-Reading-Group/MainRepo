virtual NS_DECL_ISUPPORTS void mozilla::dom::BodyRule::MapRuleInfoInto ( nsRuleData *  aRuleData) [virtual]

|nsIStyleRule::MapRuleInfoInto| is a request to copy all stylistic data represented by the rule that: + are relevant for any structs in |aRuleData->mSIDs| (style struct ID bits) + are not already filled into the data struct into the appropriate data struct in |aRuleData|. It is important that only empty data are filled in, since the rule tree is walked from highest priority rule to least, so that the walk can stop if all needed data are found. Thus overwriting non-empty data will break CSS cascading rules.
