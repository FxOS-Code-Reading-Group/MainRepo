void mozilla::layers::CompositorParent::ForceIsFirstPaint ( )

This forces the is-first-paint flag to true. This is intended to be called by the widget code when it loses its viewport information (or for whatever reason wants to refresh the viewport information). The information refresh happens because the compositor will call SetFirstPaintViewport on the next frame of composition.
