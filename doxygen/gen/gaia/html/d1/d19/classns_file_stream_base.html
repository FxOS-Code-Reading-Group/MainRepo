virtual nsresult nsFileStreamBase::DoOpen ( ) [protected, virtual]

Open the file. This is called either from MaybeOpen (during Init) or from DoPendingOpen (if DEFER_OPEN is used when initializing this stream). The default behavior of DoOpen is to open the file and save the file descriptor.

nsSafeFileOutputStream, FileQuotaStream< nsFileInputStream >, FileQuotaStream< nsFileStream >, と FileQuotaStream< nsFileOutputStream >で再定義されています。

nsresult nsFileStreamBase::DoPendingOpen ( ) [protected]

If there is a pending open, do it now. It's important for this to be inline since we do it in almost every stream API call.
