00115 /*
00116  * Perform a graceful shutdown of NSPR.  PR_Cleanup() may be called by
00117  * the primordial thread near the end of the main() function.
00118  *
00119  * PR_Cleanup() attempts to synchronize the natural termination of
00120  * process.  It does that by blocking the caller, if and only if it is
00121  * the primordial thread, until the number of user threads has dropped
00122  * to zero.  When the primordial thread returns from main(), the process
00123  * will immediately and silently exit.  That is, it will (if necessary)
00124  * forcibly terminate any existing threads and exit without significant
00125  * blocking and there will be no error messages or core files.
00126  *
00127  * PR_Cleanup() returns PR_SUCCESS if NSPR is successfully shutdown,
00128  * or PR_FAILURE if the calling thread of this function is not the
00129  * primordial thread.
00130  */
