00034 /*

00036  *
00037  * The application registry holds information about applications (uniquely
00038  * identified by id), such as which MIME types and URI schemes they are
00039  * capable of handling, whether they run in a terminal, etc.

00041  * The MIME registry holds information about MIME types, such as which
