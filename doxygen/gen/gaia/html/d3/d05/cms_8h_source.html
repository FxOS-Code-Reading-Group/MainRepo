01109 /*
01110  *  define new S/MIME content type entries
01111  *
01112  *  S/MIME uses the builtin PKCS7 oid types for encoding and decoding the
01113  *  various S/MIME content. Some applications have their own content type
01114  *  which is different from the standard content type defined by S/MIME.
01115  *
01116  *  This function allows you to register new content types. There are basically
01117  *  Two different types of content, Wrappping content, and Data.
01118  *
01119  *  For data types, All the functions below can be zero or NULL excext 
01120  *  type and is isData, which should be your oid tag and PR_FALSE respectively
01121  *
01122  *  For wrapping types, everything must be provided, or you will get encoder
01123  *  failures.
01124  *
01125  *  If NSS doesn't already define the OID that you need, you can register 
01126  *  your own with SECOID_AddEntry.
01127  * 
01128  *  Once you have defined your new content type, you can pass your new content
01129  *  type to NSS_CMSContentInfo_SetContent().
01130  * 
01131  *  If you are using a wrapping type you can pass your own data structure in 
01132  *  the ptr field, but it must contain and embedded NSSCMSGenericWrappingData 
01133  *  structure as the first element. The size you pass to 
01134  *  NSS_CMSType_RegisterContentType is the total size of your self defined 
01135  *  data structure. NSS_CMSContentInfo_GetContent will return that data 
01136  *  structure from the content info. Your ASN1Template will be evaluated 
01137  *  against that data structure.
01138  */
01139 SECStatus NSS_CMSType_RegisterContentType(SECOidTag type,
01140                           SEC_ASN1Template *asn1Template, size_t size,
01141                           NSSCMSGenericWrapperDataDestroy  destroy,
01142                           NSSCMSGenericWrapperDataCallback decode_before,
01143                           NSSCMSGenericWrapperDataCallback decode_after,
01144                           NSSCMSGenericWrapperDataCallback decode_end,
01145                           NSSCMSGenericWrapperDataCallback encode_start,
