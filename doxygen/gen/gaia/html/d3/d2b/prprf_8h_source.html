00089 **     PR_sscanf() scans the input character string, performs data

00094 **     PR_sscanf() behaves the same way as the sscanf() function in the
00095 **     Standard C Library (stdio.h), with the following exceptions:
00096 **     - PR_sscanf() handles the NSPR integer and floating point types,

00098 **       sscanf() handles the standard C types like short, int, long,
00099 **       and double.
00100 **     - PR_sscanf() has no multibyte character support, while sscanf()
00101 **       does.
00102 ** INPUTS:
00103 **     const char *buf
00104 **         a character string holding the input to scan
00105 **     const char *fmt
00106 **         the format control string for the conversions
00107 **     ...
00108 **         variable number of arguments, each of them is a pointer to
00109 **         a data object in which the converted value will be stored
00110 ** OUTPUTS: none
00111 ** RETURNS: PRInt32
00112 **     The number of values converted and stored.
00113 ** RESTRICTIONS:
00114 **    Multibyte characters in 'buf' or 'fmt' are not allowed.
00115 ***************************************************************************
