#define SEARCH_ENGINE_LOADED   "engine-loaded"

Sent when a search engine being installed from a remote plugin description file is completely loaded. This is used internally by the search service as an indication of when the engine can be added to the internal store, and therefore should not be used to detect engine availability. It is always followed by an "added" notification.
