NS_IMETHOD nsISpellChecker::GetDictionaryList ( nsTArray< nsString > *  aDictionaryList) [pure virtual]

Returns the list of strings representing the dictionaries the spellchecker supports. It was suggested that the strings returned be in the RFC 1766 format. This format looks something like <ISO 639 language code>-<ISO 3166 country code>. For example: en-US

引数:
aDictionaryListis an array of nsStrings that represent the dictionaries supported by the spellchecker.
