00033     /*
00034       Terminologies:
00035 
00036         * Symbol: single Zhuyin alphabet, e.g. "ㄊ"
00037         * Syllable: a set of symbols that made up a sound e.g. "ㄊㄞˊ",
00038           it always ended with tone symbol in Zhuyin.
00039         * Terms: a meaningful series of characters in Chinese,
00040           e.g. "台北".
00041           Possible to have one character only, e.g. "台".
00042         * Sentence: combination of terms.
00043         * Buffer: Symbols are resolved into syllables and put in the
00044           buffer for look up terms in database.
00045         * Candidates: Sentence or terms that resolved from syllables
00046           for user to select for actual output.
00047 
00048       This "Smart" Zhuyin IME aim to select terms automatically based on
00049       series of syllables user inputs, so user won't have to choose characters
00050       to build terms one by one. Still, in order to allow access to all
00051       characters, every character in our targeted range is included in the
00052       database as one character term, but with lowest score.

00873     * a given natural number
00874     * callback will be called 2^(n-1) times.

00877     * also: http://stackoverflow.com/questions/8375439

00881       var x, a, j;
00882       x = 1 << n - 1;
00883       while (x--) {
00884         a = [1];
00885         j = 0;

00887           if (x & (1 << j)) {
00888             a[a.length - 1]++;
00889           } else {
00890             a.push(1);
00891           }
00892           j++;
00893         }
00894         callback.call(this, a);
