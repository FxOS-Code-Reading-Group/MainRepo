bool nsCSSValue::IsRelativeLengthUnit ( ) const

What the spec calls relative length units is, for us, split between relative length units and pixel length units.

A "relative" length unit is a multiple of some derived metric, such as a font em-size, which itself was controlled by an input CSS length. Relative length units should not be scaled by zooming, since the underlying CSS length would already have been scaled.
