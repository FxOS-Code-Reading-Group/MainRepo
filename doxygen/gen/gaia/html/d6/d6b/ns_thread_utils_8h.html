引数:

mayWaitA boolean parameter that if "true" indicates that the method may block the calling thread to wait for a pending event.
戻り値:

NS_COM_GLUE NS_METHOD NS_ProcessPendingEvents ( nsIThread *  thread,
PRIntervalTime  timeout = PR_INTERVAL_NO_TIMEOUT 
)

Process all pending events for the given thread before returning. This method simply calls ProcessNextEvent on the thread while HasPendingEvents continues to return true and the time spent in NS_ProcessPendingEvents does not exceed the given timeout value.

引数:
threadThe thread object for which to process pending events. If null, then events will be processed for the current thread.
timeoutThe maximum number of milliseconds to spend processing pending events. Events are not pre-empted to honor this timeout. Rather, the timeout value is simply used to determine whether or not to process another event. Pass PR_INTERVAL_NO_TIMEOUT to specify no timeout.
