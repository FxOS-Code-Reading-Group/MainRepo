説明

A class representing the swizzle access pattern for a texture. Note that if the texture is an alpha-only texture then the alpha channel is substituted for other components. Any mangling to handle the r,g,b->a conversions for alpha textures is automatically included in the stage key. However, if a GrEffect uses different swizzles based on its input then it must consider that variation in its key-generation.
