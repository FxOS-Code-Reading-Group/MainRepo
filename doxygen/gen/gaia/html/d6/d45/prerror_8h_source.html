00088 /*
00089 ** Description: Localizable error code to string function.
00090 **
00091 **
00092 ** NSPR provides a mechanism for converting an error code to a
00093 ** descriptive string, in a caller-specified language.
00094 **
00095 ** Error codes themselves are 32 bit (signed) integers.  Typically,
00096 ** the high order 24 bits are an identifier of which error table the
00097 ** error code is from, and the low order 8 bits are a sequential error
00098 ** number within the table.  NSPR supports error tables whose first
00099 ** error code is not a multiple of 256, such error code assignments
00100 ** should be avoided when possible.
00101 **
00102 ** Error table 0 is defined to match the UNIX system call error table
00103 ** (sys_errlist); this allows errno values to be used directly in the
00104 ** library.  Other error table numbers are typically formed by
00105 ** compacting together the first four characters of the error table
00106 ** name.  The mapping between characters in the name and numeric
00107 ** values in the error code are defined in a system-independent
00108 ** fashion, so that two systems that can pass integral values between
00109 ** them can reliably pass error codes without loss of meaning; this
00110 ** should work even if the character sets used are not the
00111 ** same. (However, if this is to be done, error table 0 should be
00112 ** avoided, since the local system call error tables may differ.)
00113 **
00114 ** Libraries defining error codes need only provide a table mapping
00115 ** error code numbers to names and default English descriptions,
00116 ** calling a routine to install the table, making it ``known'' to NSPR
00117 ** library.  Once installed, a table may not be removed.  Any error
00118 ** code the library generates can be converted to the corresponding
00119 ** error message.  There is also a default format for error codes
00120 ** accidentally returned before making the table known, which is of
00121 ** the form "unknown code foo 32", where "foo" would be the name of
00122 ** the table.
00123 **
00124 ** Normally, the error code conversion routine only supports the
00125 ** languages "i-default" and "en", returning the error-table-provided
00126 ** English description for both languages.  The application may
00127 ** provide a localization plugin, allowing support for additional
00128 ** languages.
00129 **
00130 **/
