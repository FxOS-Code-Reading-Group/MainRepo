00007 /*
00008  * nsINodeInfo is an interface to node info, such as name, prefix, namespace
00009  * ID and possibly other data that is shared between nodes (elements
00010  * and attributes) that have the same name, prefix and namespace ID within
00011  * the same document.
00012  *
00013  * nsNodeInfoManager's are internal objects that manage a list of
00014  * nsINodeInfo's, every document object should hold a strong reference to
00015  * a nsNodeInfoManager and every nsINodeInfo also holds a strong reference
00016  * to their owning manager. When a nsINodeInfo is no longer used it will
00017  * automatically remove itself from its owner manager, and when all
00018  * nsINodeInfo's have been removed from a nsNodeInfoManager and all external
00019  * references are released the nsNodeInfoManager deletes itself.
00020  *
00021  * -- jst@netscape.com
00022  */
