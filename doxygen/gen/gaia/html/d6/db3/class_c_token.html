説明

Token objects represent sequences of characters as they are consumed from the input stream (URL). While they're pretty general in nature, we use subclasses (found in nsHTMLTokens.h) to define <start>, </end>, <text>, <comment>, <&entity>, <newline>, and <whitespace> tokens.
