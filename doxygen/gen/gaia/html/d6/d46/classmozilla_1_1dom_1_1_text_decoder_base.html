Decodes incoming byte stream of characters in charset indicated by encoding.

The encoding algorithm state is reset if aOptions.mStream is not set.

If the fatal flag is set then a decoding error will throw EncodingError. Else the decoder will return a decoded string with replacement character(s) for unidentified character(s).

引数:
aView,incomingbyte stream of characters to be decoded to to UTF-16 code points.
