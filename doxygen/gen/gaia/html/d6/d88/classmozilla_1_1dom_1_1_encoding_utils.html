static bool mozilla::dom::EncodingUtils::FindEncodingForLabel ( const nsACString &  aLabel,
nsACString &  aOutEncoding 
) [static]

Implements get an encoding algorithm from Encoding spec. http://encoding.spec.whatwg.org/#concept-encoding-get Given a label, this function returns the corresponding encoding or a false. The returned name may not be lowercased due to compatibility with our internal implementations.

引数:
aLabel,incominglabel describing charset to be decoded.
aRetEncoding,returningcorresponding encoding for label.
戻り値:
false if no encoding was found for label. true if valid encoding found.

template<class T >
static void mozilla::dom::EncodingUtils::TrimSpaceCharacters ( T &  aString) [static]

Remove any leading and trailing space characters, following the definition of space characters from Encoding spec. http://encoding.spec.whatwg.org/#terminology Note that nsAString::StripWhitespace() doesn't exactly match the definition. It also removes all matching chars in the string, not just leading and trailing.
