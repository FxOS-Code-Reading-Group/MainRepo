enum mozilla::gl::GLLibraryEGL::EGLExtensions

Known GL extensions that can be queried by IsExtensionSupported. The results of this are cached, and as such it's safe to use this even in performance critical code. If you add to this array, remember to add to the string names in GLContext.cpp.
