00006 /*
00007  * This file contains the list of event names that are exposed via IDL
00008  * on various objects.  It is designed to be used as inline input to
00009  * various consumers through the magic of C preprocessing.
00010  *
00011  * Each entry consists of 4 pieces of information:
00012  * 1) The name of the event
00013  * 2) The event ID (see nsGUIEvent.h)
00014  * 3) The event type (see the EventNameType enum in nsContentUtils.h)
00015  * 4) The event struct type for this event.
00016  * Items 2-4 might be empty strings for events for which they don't make sense.
00017  *
00018  * Event names that are exposed as content attributes on HTML elements
00019  * and as IDL attributes on Elements, Documents and Windows and have
00020  * no forwarding behavior should be enclosed in the EVENT macro.
00021  *
00022  * Event names that are exposed as content attributes on HTML elements
00023  * and as IDL attributes on Elements, Documents and Windows and are
00024  * forwarded from <body> and <frameset> to the Window should be
00025  * enclosed in the FORWARDED_EVENT macro.  If this macro is not
00026  * defined, it will be defined to be equivalent to EVENT.
00027  *
00028  * Event names that are exposed as IDL attributes on Windows only
00029  * should be enclosed in the WINDOW_ONLY_EVENT macro.  If this macro
00030  * is not defined, it will be defined to the empty string.
00031  *
00032  * Event names that are exposed as content and IDL attributes on
00033  * <body> and <frameset>, which forward them to the Window, and are
00034  * exposed as IDL attributes on the Window should be enclosed in the
00035  * WINDOW_EVENT macro.  If this macro is not defined, it will be
00036  * defined to be equivalent to WINDOW_ONLY_EVENT.
00037  *
00038  * Touch-specific event names should be enclosed in TOUCH_EVENT.  They
00039  * are otherwise equivalent to those enclosed in EVENT.  If
00040  * TOUCH_EVENT is not defined, it will be defined to the empty string.
00041  *
00042  * Event names that are only exposed as IDL attributes on Documents
00043  * should be enclosed in the DOCUMENT_ONLY_EVENT macro.  If this macro is
00044  * not defined, it will be defined to the empty string.
00045  *
00046  * Event names that are not exposed as IDL attributes at all should be
00047  * enclosed in NON_IDL_EVENT.  If NON_IDL_EVENT is not defined, it
00048  * will be defined to the empty string.
00049  *
00050  * If you change which macros event names are enclosed in, please
00051  * update the tests for bug 689564 and bug 659350 as needed.
00052  */
