define ( 'net'  )

Make our TCPSocket implementation look like node's net library.

We make sure to support:

Attributes:

  • encrypted (false, this is not the tls byproduct)
  • destroyed

Methods:

  • setKeepAlive(Boolean)
  • write(Buffer)
  • end

Events:

  • "connect"
  • "close"
  • "end"
  • "data"
  • "error"
