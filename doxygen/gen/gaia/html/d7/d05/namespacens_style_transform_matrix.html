Given an nsCSSValueList containing -moz-transform functions, returns a matrix containing the value of those functions.

引数:
aDataThe nsCSSValueList containing the transform functions
aContextThe style context, used for unit conversion.
aPresContextThe presentation context, used for unit conversion.
aCanStoreInRuleTreeSet to false if the result cannot be cached in the rule tree, otherwise untouched.
aBoundsThe frame's bounding rectangle.
aAppUnitsPerMatrixUnitThe number of app units per device pixel.

aContext and aPresContext may be null if all of the (non-percent) length values in aData are already known to have been converted to eCSSUnit_Pixel (as they are in an nsStyleAnimation::Value)
