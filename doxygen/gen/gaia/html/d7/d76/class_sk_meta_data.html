typedef void*(* SkMetaData::PtrProc)(void *ptr, bool doRef)

Used to manage the life-cycle of a ptr in the metadata. This is option in setPtr, and is only invoked when either copying one metadata to another, or when the metadata is destroyed.

setPtr(name, ptr, proc) { fPtr = proc(ptr, true); }

copy: A = B { A.fPtr = B.fProc(B.fPtr, true); }

~SkMetaData { fProc(fPtr, false); }
