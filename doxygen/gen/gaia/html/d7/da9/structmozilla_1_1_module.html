GetFactoryProcPtr mozilla::Module::getFactoryProc

When the component manager tries to get the factory for a CID, it first checks for this module-level getfactory callback. If this function is not implemented, it checks the CIDEntry getfactory callback. If that is also NULL, a generic factory is generated using the CIDEntry constructor callback which must be non-NULL.
