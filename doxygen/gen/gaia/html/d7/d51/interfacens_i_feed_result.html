attribute boolean nsIFeedResult::bozo

The Feed parser will set the bozo bit when a feed triggers a fatal error during XML parsing. There may be entries and feed metadata that were parsed before the error. Thanks to Tim Bray for suggesting this terminology. <http://www.tbray.org/ongoing/When/200x/2004/01/11/PostelPilgrim>

An XSLT stylesheet available to transform the source of the feed. Some feeds include this information in a processing instruction. It's generally intended for clients with specific feed capabilities.
