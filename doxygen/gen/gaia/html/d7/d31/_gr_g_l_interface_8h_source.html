00099 /*
00100  * GrContext uses the following interface to make all calls into OpenGL. When a
00101  * GrContext is created it is given a GrGLInterface. The interface's function
00102  * pointers must be valid for the OpenGL context associated with the GrContext.
00103  * On some platforms, such as Windows, function pointers for OpenGL extensions
00104  * may vary between OpenGL contexts. So the caller must be careful to use a
00105  * GrGLInterface initialized for the correct context. All functions that should
00106  * be available based on the OpenGL's version and extension string must be
00107  * non-NULL or GrContext creation will fail. This can be tested with the
00108  * validate() method when the OpenGL context has been made current.
00109  */
