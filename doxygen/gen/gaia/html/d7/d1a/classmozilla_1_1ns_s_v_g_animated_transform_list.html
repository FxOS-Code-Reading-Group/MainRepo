Class nsSVGAnimatedTransformList

This class is very different to the SVG DOM interface of the same name found in the SVG specification. This is a lightweight internal class - see SVGAnimatedTransformList for the heavier DOM class that wraps instances of this class and implements the SVG specification's SVGAnimatedTransformList DOM interface.

Except where noted otherwise, this class' methods take care of keeping the appropriate DOM wrappers in sync (see the comment in SVGAnimatedTransformList::InternalBaseValListWillChangeTo) so that their consumers don't need to concern themselves with that.
