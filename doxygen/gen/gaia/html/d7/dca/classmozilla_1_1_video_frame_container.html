説明

This object is used in the decoder backend threads and the main thread to manage the "current video frame" state. This state includes timing data and an intrinsic size (see below). This has to be a thread-safe object since it's accessed by resource decoders and other off-main-thread components. So we can't put this state in the media element itself ... well, maybe we could, but it could be risky and/or confusing.
