説明

A view on a YCbCr image stored with its metadata in a blob of memory. It is only meant as a convenience to access the image data, and does not own the data. The instance can live on the stack and used as follows:

const YCbCrImage& yuv = sharedImage.get_YCbCrImage(); YCbCrImageDataDeserializer deserializer(yuv.data().get<uint8_t>()); if (!deserializer.IsValid()) { // handle error } size = deserializer.GetYSize(); // work with the data, etc...
