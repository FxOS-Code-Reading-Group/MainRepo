readonly attribute AUTF8String nsINetAddr::address
戻り値:
Either the IP address (FAMILY_INET, FAMILY_INET6) or the path (FAMILY_LOCAL) in string form. IP addresses are in the format produced by mozilla::net::NetAddrToString.

Note: Paths for FAMILY_LOCAL may have length limitations which are implementation dependent and not documented as part of this interface.
