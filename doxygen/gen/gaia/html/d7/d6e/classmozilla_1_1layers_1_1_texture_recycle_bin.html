説明

A RecycleBin is owned by an ImageLayer. We store textures in it that we want to recycle from one image to the next. It's a separate object from ImageContainer because images need to store a strong ref to their RecycleBin and we must avoid creating a reference loop between an ImageContainer and its active image.
