in nsIObserver  aObserver

Schedule a check to see if an update is available.

This will not update or make any changes to the appcache. It only notifies the observer to indicate whether the manifest has changed on the server (or not): a changed manifest means that an update is available.

For arguments see nsIOfflineCacheUpdate.initForUpdateCheck() method description.
