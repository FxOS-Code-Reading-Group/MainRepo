説明

template<typename T>
class SkTScopedPtr< T >

A SkTScopedPtr<T> is like a T*, except that the destructor of SkTScopedPtr<T> automatically deletes the pointer it holds (if any). That is, SkTScopedPtr<T> owns the T object that it points to. Like a T*, a SkTScopedPtr<T> may hold either NULL or a pointer to a T object. Also like T*, SkTScopedPtr<T> is thread-compatible, and once you dereference it, you get the threadsafety guarantees of T.

The size of a SkTScopedPtr is small: sizeof(SkTScopedPtr<T>) == sizeof(T*)
