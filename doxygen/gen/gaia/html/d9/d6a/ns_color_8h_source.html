00036 // Fast approximate division by 255. It has the property that
00037 // for all 0 <= n <= 255*255, FAST_DIVIDE_BY_255(n) == n/255.
00038 // But it only uses two adds and two shifts instead of an 
00039 // integer division (which is expensive on many processors).
