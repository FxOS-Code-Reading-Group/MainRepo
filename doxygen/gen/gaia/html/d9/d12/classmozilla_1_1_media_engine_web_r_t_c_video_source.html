説明

The WebRTC implementation of the MediaEngine interface.

On B2G platform, member data may accessed from different thread after construction:

MediaThread: mState, mImage, mWidth, mHeight, mCapability, mPrefs, mDeviceName, mUniqueId, mInitDone, mSources, mImageContainer, mSources, mState, mImage, mLastCapture

MainThread: mDOMCameraControl, mCaptureIndex, mCameraThread, mWindowId, mCameraManager, mNativeCameraControl, mPreviewStream, mState, mLastCapture, mWidth, mHeight

Where mWidth, mHeight, mImage are protected by mMonitor mState, mLastCapture is protected by mCallbackMonitor Other variable is accessed only from single thread
