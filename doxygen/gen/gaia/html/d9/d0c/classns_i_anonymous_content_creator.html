virtual nsresult nsIAnonymousContentCreator::CreateAnonymousContent ( nsTArray< ContentInfo > &  aElements) [pure virtual]

Creates "native" anonymous content and adds the created content to the aElements array. None of the returned elements can be nullptr.

If the anonymous content creator sets the editable flag on some of the elements that it creates, the flag will be applied to the node upon being bound to the document.

virtual nsIFrame* nsIAnonymousContentCreator::CreateFrameFor ( nsIContent *  aContent) [virtual]

Implementations can override this method to create special frames for the anonymous content returned from CreateAnonymousContent. By default this method returns nullptr, which means the default frame is created.
