Extract the charset parameter location and value from a content-type header.

引数:
aTypeHeaderthe header string to parse
[out]aCharsetthe charset parameter specified in the header, if any.
[out]aCharsetStartindex of the start of the charset parameter (the ';' separating it from what came before) in aTypeHeader. If this function returns false, this argument will still be set, to the index of the location where a new charset should be inserted.
[out]aCharsetEndindex of the end of the charset parameter (the ';' separating it from what comes after, or the end of the string) in aTypeHeader. If this function returns false, this argument will still be set, to the index of the location where a new charset should be inserted.
戻り値:
whether a charset parameter was found. This can be false even in cases when parseContentType would claim to have a charset, if the type that won out does not have a charset parameter specified.
