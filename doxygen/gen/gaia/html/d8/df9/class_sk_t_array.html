説明

template<typename T, bool MEM_COPY>
class SkTArray< T, MEM_COPY >

When MEM_COPY is true T will be bit copied when moved. When MEM_COPY is false, T will be copy constructed / destructed. In all cases T's constructor will be called on allocation, and its destructor will be called from this object's destructor.
