00013  * nsSMILTimeValue class
00014  *
00015  * A tri-state time value.
00016  *
00017  * First a quick overview of the SMIL time data types:
00018  *
00019  * nsSMILTime          -- a timestamp in milliseconds.
00020  * nsSMILTimeValue     -- (this class) a timestamp that can take the additional
00021  *                        states 'indefinite' and 'unresolved'
00022  * nsSMILInstanceTime  -- an nsSMILTimeValue used for constructing intervals. It
00023  *                        contains additional fields to govern reset behavior
00024  *                        and track timing dependencies (e.g. syncbase timing).
00025  * nsSMILInterval      -- a pair of nsSMILInstanceTimes that defines a begin and
00026  *                        an end time for animation.
00027  * nsSMILTimeValueSpec -- a component of a begin or end attribute, such as the
00028  *                        '5s' or 'a.end+2m' in begin="5s; a.end+2m". Acts as
00029  *                        a broker between an nsSMILTimedElement and its
00030  *                        nsSMILInstanceTimes by generating new instance times
00031  *                        and handling changes to existing times.
00032  *
00033  * Objects of this class may be in one of three states:
00034  *
00035  * 1) The time is resolved and has a definite millisecond value
00036  * 2) The time is resolved and indefinite
00037  * 3) The time is unresolved
00038  *
00039  * In summary:
00040  *
00041  * State      | GetMillis       | IsDefinite | IsIndefinite | IsResolved
00042  * -----------+-----------------+------------+--------------+------------
00043  * Definite   | nsSMILTimeValue | true       | false        | true
00044  * -----------+-----------------+------------+--------------+------------
00045  * Indefinite | --              | false      | true         | true
00046  * -----------+-----------------+------------+--------------+------------
00047  * Unresolved | --              | false      | false        | false
00048  *
00049  */
