00005 /*
00006 ** File:        nsrwlock.h
00007 ** Description: API to basic reader-writer lock functions of NSS.
00008 **  These are re-entrant reader writer locks; that is,
00009 **  If I hold the write lock, I can ask for it and get it again.
00010 **  If I hold the write lock, I can also ask for and get a read lock.
00011 **      I can then release the locks in any order (read or write).
00012 **  I must release each lock type as many times as I acquired it.
00013 **  Otherwise, these are normal reader/writer locks.
00014 **
00015 ** For deadlock detection, locks should be ranked, and no lock may be aquired
00016 ** while I hold a lock of higher rank number.
00017 ** If you don't want that feature, always use NSS_RWLOCK_RANK_NONE.
00018 ** Lock name is for debugging, and is optional (may be NULL)
00019 **/
