nsIDOMRange nsIFind::Find ( in wstring  aPatText,
in nsIDOMRange  aSearchRange,
in nsIDOMRange  aStartPoint,
in nsIDOMRange  aEndPoint 
)

Find some text in the current context. The implementation is responsible for performing the find and highlighting the text.

引数:
aPatTextThe text to search for.
aSearchRangeA Range specifying domain of search.
aStartPointA Range specifying search start point. If not collapsed, we'll start from end (forward) or start (backward).
aEndPointA Range specifying search end point. If not collapsed, we'll end at end (forward) or start (backward).
戻り値:
Arange spanning the match that was found (or null).
