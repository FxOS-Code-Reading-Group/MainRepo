フレンド

class TiledLayerBuffer< BasicTiledLayerBuffer, BasicTiledLayerTile >

説明

Provide an instance of TiledLayerBuffer backed by image surfaces. This buffer provides an implementation to ValidateTile using a thebes callback and can support painting using a single paint buffer which is much faster then painting directly into the tiles.
