説明

ContainerLayer that refers to a "foreign" layer tree, through an ID. Usage of RefLayer looks like

Construction phase: allocate ID for layer subtree create RefLayer, SetReferentId(ID)

Composition: look up subtree for GetReferentId() ConnectReferentLayer(subtree) compose ClearReferentLayer()

Clients will usually want to Connect/Clear() on each transaction to avoid difficulties managing memory across multiple layer subtrees.
