説明

The nsITextServicesDocument presents the document in as a bunch of flattened text blocks. Each text block can be retrieved as an nsString (array of characters).

NS_IMETHOD nsITextServicesDocument::DeleteSelection ( ) [pure virtual]

Deletes the text selected by SetSelection(). Calling DeleteSelection with nothing selected, or with a collapsed selection (cursor) does nothing and returns NS_OK.

NS_IMETHOD nsITextServicesDocument::IsDone ( bool *  aIsDone) [pure virtual]

IsDone() will always set aIsDone == false unless the document contains no text, PrevBlock() was called while the document was already pointing to the first text block in the document, or NextBlock() was called while the document was already pointing to the last text block in the document.

引数:

NS_IMETHOD nsITextServicesDocument::NextBlock ( ) [pure virtual]

Tells the document to point to the text block after the current one. This method will return NS_OK, even if there is no next block. Callers should call IsDone() to check if we have gone beyond the last text block in the document.

NS_IMETHOD nsITextServicesDocument::PrevBlock ( ) [pure virtual]

Tells the document to point to the text block before the current one. This method will return NS_OK, even if there is no previous block. Callers should call IsDone() to check if we have gone beyond the first text block in the document.

NS_IMETHOD nsITextServicesDocument::ScrollSelectionIntoView ( ) [pure virtual]

Scrolls the document so that the current selection is visible.

NS_IMETHOD nsITextServicesDocument::SetExtent ( nsIDOMRange *  aDOMRange) [pure virtual]

Sets the range/extent over which the text services document will iterate. Note that InitWithEditor() should have been called prior to calling this method. If this method is never called, the text services defaults to iterating over the entire document.

引数:
aDOMRangeis the range to use. aDOMRange must point to a valid range object.
