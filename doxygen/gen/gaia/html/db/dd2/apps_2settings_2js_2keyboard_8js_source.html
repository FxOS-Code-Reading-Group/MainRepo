00007  * The script depends shared/js/keyboard_helper.js. KeyboardHelper helps on the
00008  * following things:
00009  * - Get all installed keyboard apps and layouts.
00010  * - Enable or disable keyboard layouts.
00011  * - Notify keyboard layout changes via the 'keyboardsrefresh' event.
00012  *
00013  * This javascript file defines the following modules:
00014  * - KeyboardContext
00015  *   KeyboardContext provides lists of all installed keyboards and enabled
00016  *   layouts. The lists are kept updated based on the events from
00017  *   KeyboardHelper. KeyboardContext handles only data and does not involve in
00018  *   any UI logic.
00019  *
00020  * - Panel
00021  *   Panel is designed to be initialized with a panel ID, and reflects if it is
00022  *   visible to users via the `visible` property.
00023  *
00024  * - KeyboardPanel
00025  * - EnabledLayoutsPanel
00026  * - InstalledLayoutsPanel
00027  *   In the panels we initialize a ListView with the data provided by
00028  *   KeyboardContext. Templates for generating UI elements are also defined
00029  *   here. The three panel used Panel for observing the visibility change. Which
00030  *   is for avoiding unnecessary UI changes when the panels are not visible to
00031  *   users.
00032  */
