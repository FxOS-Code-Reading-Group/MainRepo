Called to remove an individual cookie from the cookie list, specified by host, name, and path. If the cookie cannot be found, no exception is thrown. Typically, the arguments to this method will be obtained directly from the desired nsICookie object.

引数:
aHostThe host or domain for which the cookie was set.
参照:
nsICookieManager2::add for a description of acceptable host strings. If the target cookie is a domain cookie, a leading dot must be present.
引数:
aNameThe name specified in the cookie
aPathThe path for which the cookie was set
aBlockedIndicates if cookies from this host should be permanently blocked
