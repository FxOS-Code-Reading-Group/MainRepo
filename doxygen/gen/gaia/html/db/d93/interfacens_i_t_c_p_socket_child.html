void nsITCPSocketChild::setSocketAndWindow ( in nsITCPSocketInternal  socket,
in jsval  windowVal 
)

Initialize the TCP socket on the child side for IPC. It is called from the child side, which is generated in receiving a notification of accepting any open request on the parent side. We use single implementation that works on a child process as well as in the single process model.

引数:
socketThe TCP socket on the child side. This instance is connected with the child IPC side of the IPC bridge.
windowValThe window object on the child side to create data as "jsval" for deserialization.
