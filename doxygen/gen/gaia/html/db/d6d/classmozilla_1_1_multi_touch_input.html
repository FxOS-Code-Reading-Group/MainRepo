説明

Similar to nsTouchEvent, but for use off-main-thread. Also only stores a screen touch point instead of the many different coordinate spaces nsTouchEvent stores its touch point in. This includes a way to initialize itself from an nsTouchEvent by copying all relevant data over. Note that this copying from nsTouchEvent functionality can only be used on the main thread.

Stores an array of SingleTouchData.
