This subclass of shader returns the coposition of two other shaders, combined by a xfermode.

SkXfermode *  mode = NULL

Create a new compose shader, given shaders A, B, and a combining xfermode mode. When the xfermode is called, it will be given the result from shader A as its "dst", and the result of from shader B as its "src". mode->xfer32(sA_result, sB_result, ...)

引数:
shaderAThe colors from this shader are seen as the "dst" by the xfermode
shaderBThe colors from this shader are seen as the "src" by the xfermode
modeThe xfermode that combines the colors from the two shaders. If mode is null, then SRC_OVER is assumed.

Assuming setContext returned true, endContext() will be called when the draw using the shader has completed. It is an error for setContext to be called twice w/o an intervening call to endContext().
