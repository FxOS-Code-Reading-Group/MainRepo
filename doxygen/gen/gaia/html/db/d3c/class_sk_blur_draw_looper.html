virtual bool SkBlurDrawLooper::next ( SkCanvas *  ,
SkPaint *  paint 
) [virtual]

Called in a loop (after init()). Each time true is returned, the object is drawn (possibly with a modified canvas and/or paint). When false is finally returned, drawing for the object stops.

On each call, the paint will be in its original state, but the canvas will be as it was following the previous call to next() or init().

The implementation must ensure that, when next() finally returns false, that the canvas has been restored to the state it was initially, before init() was first called.
