Parse aBuffer into a media list |aMediaList|, which must be non-null, replacing its current contents. If aHTMLMode is true, parse according to HTML rules, with commas as the most important delimiter. Otherwise, parse according to CSS rules, with parentheses and strings more important than commas. |aURL| and |aLineNumber| are used for error reporting.

uint32_t  aLineNumber,

Parse aInput into the stylesheet that was previously set by calling SetStyleSheet. Calling this method without calling SetStyleSheet first is an error.

引数:
aInputthe data to parse
aSheetURLthe URI to use as the sheet URI (for error reporting). This must match the URI of the sheet passed to SetStyleSheet.
aBaseURIthe URI to use for relative URI resolution
aSheetPrincipalthe principal of the stylesheet. This must match the principal of the sheet passed to SetStyleSheet.
aLineNumberthe line number of the first line of the sheet.
aAllowUnsafeRulessee aEnableUnsafeRules in mozilla::css::Loader::LoadSheetSync
