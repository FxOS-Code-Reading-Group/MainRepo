説明

A cross-platform interface that supports hyperlink-specific properties and methods. Anchors, image maps, xul:labels with class="text-link" implement this interface.

readonly attribute long nsIAccessibleHyperLink::anchorCount

The numbber of anchors within this Hyperlink. Is normally 1 for anchors. This anchor is, for example, the visible output of the html:a tag. With an Image Map, reflects the actual areas within the map.

readonly attribute boolean nsIAccessibleHyperLink::selected

Determines whether the element currently has the focus, e. g. after returning from the destination page.

覚え書き:
ARIA links can only be focused if they have the tabindex attribute set. Also, state_focused should then be set on the accessible for this link.
