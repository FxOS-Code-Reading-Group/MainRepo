GrEffect subclasses should insert this macro in their implementation file. They must then also implement this static function: GrEffect* TestCreate(SkMWCRandom*, GrContext*, const GrDrawTargetCaps&, GrTexture* dummyTextures[2]); dummyTextures[] are valid textures that can optionally be used to construct GrTextureAccesses. The first texture has config kSkia8888_GrPixelConfig and the second has kAlpha_8_GrPixelConfig. TestCreate functions are also free to create additional textures using the GrContext.
