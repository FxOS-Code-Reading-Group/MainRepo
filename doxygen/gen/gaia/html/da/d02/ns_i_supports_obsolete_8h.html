#define NS_FREE_XPCOM_ISUPPORTS_POINTER_ARRAY (   size,
  array 
)    NS_FREE_XPCOM_POINTER_ARRAY((size), (array), NS_IF_RELEASE)

Macro to free an array of pointers to nsISupports (or classes derived from it). A convenience wrapper around NS_FREE_XPCOM_POINTER_ARRAY.

Note that if you know that none of your nsISupports pointers are going to be 0, you can gain a bit of speed by calling NS_FREE_XPCOM_POINTER_ARRAY directly and using NS_RELEASE as your free function.

引数:
sizeNumber of elements in the array. If not a constant, this should be a int32_t. Note that this means this macro will not work if size >= 2^31.
arrayThe array to be freed.
