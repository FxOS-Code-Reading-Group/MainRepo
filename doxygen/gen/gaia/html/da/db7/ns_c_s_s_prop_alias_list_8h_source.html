00007  * a list of all CSS property aliases with data about them, for preprocessing
00008  */
00009 
00010 /******
00011 
00012   This file contains the list of all CSS properties that are just
00013   aliases for other properties (e.g., for when we temporarily continue
00014   to support a prefixed property after adding support for its unprefixed
00015   form).  It is designed to be used as inline input through the magic of
00016   C preprocessing.  All entries must be enclosed in the appropriate
00017   CSS_PROP_ALIAS macro which will have cruel and unusual things done to
00018   it.
00019 
00020   The arguments to CSS_PROP_ALIAS are:
00021 
00022   -. 'aliasname' entries represent a CSS property name and *must* use
00023   only lowercase characters.
00024 
00025   -. 'id' should be the same as the 'id' field in nsCSSPropList.h for
00026   the property that 'aliasname' is being aliased to.
00027 
00028   -. 'method' is the CSS2Properties property name.  Unlike
00029   nsCSSPropList.h, prefixes should just be included in this file (rather
00030   than needing the CSS_PROP_DOMPROP_PREFIXED(prop) macro).
00031 
00032   -. 'pref' is the name of a pref that controls whether the property
00033   is enabled.  The property is enabled if 'pref' is an empty string,
00034   or if the boolean property whose name is 'pref' is set to true.
00035 
00036  ******/
