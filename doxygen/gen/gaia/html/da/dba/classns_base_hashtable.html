template<class KeyClass, class DataType, class UserDataType>
class nsBaseHashtable< KeyClass, DataType, UserDataType >

templated hashtable for simple data types This class manages simple data types that do not need construction or destruction.

引数:
KeyClassa wrapper-class for the hashtable key, see nsHashKeys.h for a complete specification.
DataTypethe datatype stored in the hashtable, for example, uint32_t or nsCOMPtr. If UserDataType is not the same, DataType must implicitly cast to UserDataType
UserDataTypethe user sees, for example uint32_t or nsISupports*
