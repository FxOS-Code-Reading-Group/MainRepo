template<typename T>
BlockRefType* SkAutoTUnref< T >::operator-> ( ) const

SkAutoTUnref assumes ownership of the ref. As a result, it is an error for the user to ref or unref through SkAutoTUnref. Therefore SkAutoTUnref::operator-> returns BlockRef<T>*. This prevents use of skAutoTUnrefInstance->ref() and skAutoTUnrefInstance->unref().
