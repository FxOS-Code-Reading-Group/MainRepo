void mozWritePoison ( void *  aPtr,
size_t  aSize 
)

Overwrite the memory block of aSize bytes at aPtr with the poison value. aPtr MUST be aligned at a sizeof(uintptr_t) boundary. Only an even number of sizeof(uintptr_t) bytes are overwritten, the last few bytes (if any) is not overwritten.
