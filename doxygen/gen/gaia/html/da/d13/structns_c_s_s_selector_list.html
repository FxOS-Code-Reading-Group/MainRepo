nsCSSSelector* nsCSSSelectorList::AddSelector ( PRUnichar  aOperator)

Create a new selector and push it onto the beginning of |mSelectors|, setting its |mNext| to the current value of |mSelectors|. If there is an earlier selector, set its |mOperator| to |aOperator|; else |aOperator| must be PRUnichar(0). Returns the new selector. The list owns the new selector. The caller is responsible for updating |mWeight|.
